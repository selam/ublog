ublog
=====

ublog, plasmoid widget for twitter 

<img src="http://i.imgur.com/uT5o05d.png">



### installation ###
```
git clone https://github.com/selam/ublog.git
plasmapkg -i ublog
```
after installation you can add to your desktop as other widget

### uninstall ###

```
plasmapkg -r ublog
```
